# Field Probe — Forced Open Source Experiment

## 核心假说

**行为即结构**：只观察模型的输出行为（logits/概率分布），能否反推其内部势场结构？

如果成立 → 闭源模型无法隐藏其内部结构，因为每次输出都在广播自己的"势场形状"。

## 实验进展

| 版本 | 样本 | 方法 | 最佳 r | Bootstrap CI | 结论 |
|------|------|------|--------|-------------|------|
| V1 | 25条/367tokens | Logit PCA vs 激活PC1 | 0.280 | - | 熵比logit更好(0.426) |
| V2 | 25条/367tokens | 11特征Ridge | 0.625* | - | 过拟合，R²=-0.46 |
| V3 | 100条/1539tokens | 3独立特征+Bootstrap | 0.650(L11) | [0.619,0.675] | **初步成立** |

*V2在训练集上的值，CV验证失败

## 核心发现

1. **最强行为特征是输出熵**（r=0.42~0.58），不是logit方向
   - 模型越不确定（熵高）→ 内部信息密度轴PC1越低
   - 即：输出不确定性直接编码了内部场强

2. **越深的层越容易从行为读出**（L0:r=0.36 → L11:r=0.65）
   - 早期层：探索期，结构丰富但与输出关系远
   - 晚期层：已经在计算输出，内外高度绑定

3. **Bootstrap置信区间稳定**：L11 [0.619, 0.675]，信号真实

## 与UFL框架的关系

- 势场PC1 = 信息密度轴（已在GPT-2/Qwen验证）
- 对齐是调制不是重建（Qwen d爆炸式增长）
- 行为泄露结构 → "强行开源"：闭源模型每次输出都在暴露势场形状

## 文件

- `field_probe_v1.py` — 基础实验，发现熵是最强单变量
- `field_probe_v2.py` — 11特征组合，发现过拟合问题
- `field_probe_v3.py` — 100条文本+3独立特征+Bootstrap验证
